Our battle tested, scablable and performance tweaked Red5 Media Server is perfect for gaming development or streaming media delivery. The above video is H264 encoded and streaming via RTMP via your new Red5 Media Server using video.js.

  1. Login via ssh with the key you used to start your instance and username 'ubuntu' i.e.
    ssh -i mykey.pem ubuntu@${pageContext.request.getServerName()}
  2. Interact with the Red5 upstart service:
    sudo service red5 status|start|stop
    This will work directly with the red5 service.
